Design of the Brand

The JumpStart project is a YouthSpark initiative from Microsoft. The project takes place in Puerto Rico and is focused on providing tools for young people. Also, during the project the participants develop skills around software development, marketing, design and entrepreneurship. The purpose of the project is to create a new generation of entrepreneurs to innovate for a better Puerto Rico.
